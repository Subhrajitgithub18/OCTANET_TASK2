# 📝 ToDo List - Octanet Internship Task 2

This is a fully functional ToDo List web app built using **HTML**, **CSS**, and **JavaScript**. It allows users to:

- Add tasks
- Mark them as complete
- Delete tasks
- Save to local storage

## 🚀 Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)
